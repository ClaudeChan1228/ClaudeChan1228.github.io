ROE 与 ROIC 的区别——如何看穿高杠杆的伪装

本文最后更新于 2026-06-19

ROE 可以靠借钱放大，ROIC 不能。真正优秀的企业，两者都应该很高，而且差距不大。

理解了 ROE 和 ROIC 的区别，就读懂了一半的企业财务质量。

一、ROE：Return on Equity（净资产收益率）

1.1 英文拆解

单词 含义
Return 回报、收益
on 基于……
Equity 股东权益（净资产）

因此：

Return on Equity = 股东权益的回报率

也可以理解为：股东投入资本的收益率。

1.2 ROE 衡量什么？

ROE 回答的是一个非常简单的问题：

股东投入的钱，一年能赚回来多少？

意味着：股东每投入 100 元，公司一年能赚 20 元。

  • Assets：企业拥有的一切资产

ROE 看的不是企业全部的钱，而是属于股东自己的钱，创造了多少利润。

二、ROIC：Return on Invested Capital（投入资本回报率）

2.1 英文拆解

单词 含义
Return 收益
Invested 已经投入的
Capital 资本

因此：

Return on Invested Capital = 投入资本的收益率

2.2 ROIC 衡量什么？

注意：这里不是只看股东的钱。而是所有投入企业经营的钱。

只要用于企业经营，都属于 Invested Capital。

ROIC 关注的是：企业经营所使用的全部资本创造了多少收益。

三、为什么一个叫 Equity，一个叫 Capital？

这是 ROE 和 ROIC 最核心的区别。

Equity

强调：

Who owns the money?（钱是谁的？）

答案：股东。

因此，ROE 只关注：股东的钱赚了多少。

Capital

强调：

What money is used to run the business?（企业用了哪些钱经营？）

答案：所有投入企业经营的钱。

因此，ROIC 关注：企业使用的全部资本创造了多少收益。

四、举例理解

假设一家企业：

ROE（只看股东的钱）

ROIC（看全部经营资本）

虽然 ROE 高达 30%，但企业真正利用全部资本创造收益的能力只有 10%。

高 ROE 被高负债”美化”了。

五、为什么价值投资者更看重 ROIC？

来看两家公司的对比。

A 公司：真正的经营高手

指标 数值
ROE 20%
ROIC 20%

说明：企业赚钱主要依靠经营能力，不依赖杠杆。

B 公司：高杠杆的假象

指标 数值
ROE 20%
ROIC 2%

B 公司的 ROE 看起来和 A 公司一样优秀（20%）。

但是 ROIC 只有 2% —— 企业真正的经营效率很低。ROE 只是因为大量借钱（高杠杆）而被”放大”了。

ROIC 比 ROE 更难被高负债”美化”。 这正是巴菲特和芒格等价值投资者更看重 ROIC 的原因。

六、一个奶茶店的例子

假设你开一家奶茶店：

ROE（只看自己的钱）

意思：自己投入的 50 万元，一年赚了 30%。

ROIC（看所有经营资金）

意思：企业利用 100 万元经营资金，一年创造了 15% 的收益。

30% 看起来很诱人，但它有一半是靠银行贷款撬动的。15% 更能反映你经营这家店的真实水平。

七、ROE 与 ROIC 的一句话区别

ROE — Return on Equity

关注：股东的钱赚得怎么样？

How well does management grow the shareholders’ money?

ROIC — Return on Invested Capital

关注：企业所有经营资本利用得怎么样？

How efficiently does the company use all invested capital?

八、核心对比总览

对比项 ROE ROIC
英文全称 Return on Equity Return on Invested Capital
中文 净资产收益率 投入资本回报率
关注对象 股东权益 全部投入资本
核心问题 股东的钱赚了多少？ 企业经营资本赚了多少？
是否受高负债影响 容易被高杠杆放大 不容易被高杠杆美化
更适合评价 股东投资回报 企业真实经营能力

九、如何运用 ROE + ROIC 判断企业质量？

将 ROE 和 ROIC 结合来看，可以快速识别企业的盈利质量：

情景 ROE ROIC 判断
优质企业 高 高 经营能力强，不依赖杠杆
高杠杆企业 高 低 利润靠负债放大，需警惕财务风险
低效企业 低 低 经营能力弱，缺乏投资价值
特殊阶段 低 高 可能是暂时的利润波动或大量折旧所致

优秀企业通常同时具备 ROE 持续较高、ROIC 同样较高、且两者差距不大的特征。

十、最终总结

可以用两句话记住两者的本质区别：

ROE = Return on the Owners’ Money

股东的钱赚得怎么样？

ROIC = Return on All Operating Capital

所有经营资本赚得怎么样？

换句话说：

  • ROE 回答的是：”股东赚得怎么样？”
  • ROIC 回答的是：”企业经营得怎么样？”

对于长期价值投资而言：

  • ROE 能衡量股东资本的盈利能力；
  • ROIC 更能反映企业真实的经营质量；
  • 两者结合，才能看清一家企业是在经营赚钱，还是在借钱赚钱。

如果遇到 ROE 很高、ROIC 很低 的企业，往往意味着：

企业依赖大量负债（高杠杆）提高股东收益。此时需要进一步分析其债务结构、偿债能力和财务风险——高杠杆在顺境中放大收益，在逆境中同样放大灾难。

本文为个人对 ROE 与 ROIC 两个财务指标的概念梳理与学习总结。文中提及的分析框架仅用于说明财务指标的应用方式，不构成任何投资建议。
